# 🚀 To-Do Pro

A modern and responsive To-Do application built with **Vanilla JavaScript**, focusing on clean architecture, modularization, and DOM manipulation.

---

## 🌐 Live Demo

👉 https://todo-pro-theta-woad.vercel.app/

---

## 🧠 About the Project

This project was developed to practice and demonstrate:

* JavaScript ES Modules (import/export)
* DOM manipulation
* Asynchronous programming (async/await)
* Application state management
* LocalStorage persistence

It follows a modular structure similar to real-world front-end applications.

---

## ✨ Features

* ✅ Add new tasks
* ❌ Delete tasks
* ✔️ Mark tasks as completed
* 💾 Persistent data with LocalStorage
* 📊 Task counter
* 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

* JavaScript (ES6+)
* HTML5
* CSS3

---

## 📂 Project Structure

todo-pro/
  index.html
  style.css
  app.js
  api.js
  ui.js

---

🧩 Architecture Overview

* **app.js** → Main controller (logic & events)
* **api.js** → Data layer (simulated API + localStorage)
* **ui.js** → DOM rendering and UI updates

This separation improves scalability and maintainability.
